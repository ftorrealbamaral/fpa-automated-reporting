# Project: FP&A Automated Reporting (Tasks 1–4)

# What this repo contains
# 01_file_index.ipynb: builds the file index from the transaction folder structure
# GUI_App.ipynb: GUI app that lets the user select a folder/year and generate Report 1
# src/: reusable functions (report generation + helpers)
# requirements.txt: dependencies

# How to run

# Create environment:
pip install -r requirements.txt

# Launch Jupyter:
jupyter notebook

# Open GUI_App.ipynb → Run all cells → Click Pick Folder → Scan → Generate Report 1

# Expected output
# Generated report saved into output/
# Log file saved into logs/

# Notes
# Transaction data is not included in the repo (intentionally). The app expects the same folder structure as provided in the assessment.
